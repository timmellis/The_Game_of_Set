The Game of SET

Instructions:

The object of the game is to identify a SET of 3 cards and remove them from the board.

Each card has four distinct features:

  • Number
  • Color
  • Shape
  • Shading

A SET consists of three cards on which each feature is either ALL THE SAME, or ALL DIFFERENT. For example, if each card has one item that would be considered "all the same". If the 1st card had one item, the 2nd card had two items, and the 3rd card has three items, that would be considered "all different".

Game design and layout by Tim Ellis, as part of the General Assembly: Software Engineering Intensive program.
Based on the card game "SET: The Family Game of Visual Perception", published by Set Enterprises, (c) 1991.
